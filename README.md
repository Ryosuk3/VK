#### Тестовое задание для стажировки VK - приложение видеоплеер.
Приложение загружает список роликов из api.video, показывает карточки с превью и длительностью, поддерживает pull‑to‑refresh, кэширует список и превью в Room для оффлайн‑просмотра, а по клику открывает экран воспроизведения через Media3 ExoPlayer.

####Технологии и библиотеки:

Jetpack Compose,  Navigation‑Compose, Media3 ExoPlayer, Retrofit + OkHttp + Gson для сети, Room для локального кэша, Koin для DI, Coroutines.
